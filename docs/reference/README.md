# Reference

The endpoint-level source of truth is generated by drf-spectacular.

## Live Endpoints

- Swagger UI: `/`
- OpenAPI schema: `/schema/`

## Export a Versioned Schema File

Run this command from the repository root:

```bash
python manage.py spectacular --file docs/reference/openapi.yaml --validate
```

Suggested release process:

1. Update API code and schema annotations.
2. Regenerate `docs/reference/openapi.yaml`.
3. Commit code and schema in the same pull request.
4. If behavior changed, update one or more guides in `docs/how-to` or `docs/tutorials`.
